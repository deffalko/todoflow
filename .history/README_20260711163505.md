## ✨ Функционал

- ✅ Добавление задач (Enter или кнопка)
- ✅ Удаление задач
- ✅ Отметка о выполнении
- ✅ Фильтрация (Все / Активные / Выполненные)
- ✅ Счётчик оставшихся задач
- ✅ Очистка выполненных
- ✅ Тёмная/светлая тема
- ✅ Сохранение в LocalStorage
- ✅ Анимации и уведомления

## 🛠️ Технологии

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Hot Toast
- Vite

## 🚀 Быстрый старт

```bash
git clone https://github.com/ВАШ_НИК/todoflow.git
cd todoflow
npm install
npm run dev
```
